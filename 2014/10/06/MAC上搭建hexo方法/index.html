MAC上搭建hexo方法

一:

1. 安装nvm(node.js的版本管理工具）

2.

3.

4. $ wget -q0- https://raw.github.com/creationix/nvm/master/install.sh | sh

5.

6.

7. 添加如下内容到.zshrc配置文件

8.

9.

10. [ -s “/Users/users/.nvm/nvm.sh” ] && . “/Users/users/.nvm/nvm.sh” # This loads nvm

11.

12.

13. 安装node.js(通过nvm安装)

14.

15.

16. $ nvm install 0.10

17.

18.

19. 安装hexo（通过npm安装）

20. 1

21.

22. $ npm install -g hexo

23.

24.

25. 创建hexo文件夹，并安装hexo相关组件

26.

27.

28.

29.

30. $ hexo init hexo

31. $ cd hexo

32. $ npm install

33.

二.往自己blog 上传blog

1.在hexo目录下输入：hexo new 春暖花开，就创建了一个“ 春暖花开”的文件，然后用文本编辑器编辑。

如果创建成功则会提示存在hexo/source/_posts目录下

2.hexo目录下：hexo g //将文本转换为静态网页

3.hexo d //上传至github server

4. 访问自己博客即可看到效果
