1.字符串格式化输出

－stringWithFormat

  • NSString *formaStr = [NSString stringWithFormat :”I am zhang %d,and I am %d years old!”];
  • NSLog(@”%@”,formaStr);

2.字符串长度

—-length

NSString *string01 = @”Hello,world”;
//NSLog(@”%lu”,[string01 length]);
NSUInteger _length = [string01 length]
NSLog(@”%lu”,_length);

3.字符串比较

* 3.1———只比较两个字符串是否相同，相同输出1，不同输出0
NSString string02 = @”Happy birthday!”
NSString string02 = @”Happy new year!”
isEqualToString———
BOOL result01 = [string01 isEqualToString:string02]
NSLog(@”result01 = %@”,result01);
* 3.2———不区分大小写比较
NSString string03 = @”HAPPY BIRTHDAY!”;
if([string01 caseInsenstiveCompare:string02] == NSOrderedSame){
NSLog(@"string01 is equal to string03");

}

或者：
if([string01 compare:string03 option:NSCaseInsentiveSearch == NSOrderedSame]){
NSLog(@"string01 is equal to string03");

}

4.改变字符串的大小写－－－－－

*4.1把字符串改为大写－－－uppercaseString

NSString *string04 = [string01 uppercaseString];
NSLog(@”string04 = %@”,string04);

*4.2把字符串改为小写－－－lowercaseString

NSString *string05 = [string04 lowercaseString];
NSLog(@”string05 = %@”,string05);

5.判断字符串是否含有后缀——hasSuffix—-

NSString *filename01 = @”fistname.txt”;

if([filename01 hasSuffix:@”.txt”])

    NSLog(@"filename01 has suffix with .txt");

6.判断字符串是否含有前缀——hasprefix—-

NSString *filename02 = @”studentManagement.c”;

if([filename02 hasprefix:@”student”])

    NSLog(@"filename01 has prefix with student");

7.提取字符串－－－－substringFromIndex—-从字符串的第index位置开始提取，包括第index位

NSString *string06;

string06 =［string01 substringFromIndex:3］;

NSLog(@”string06 = %@”,string06);

8.提取字符串－－－－substringToIndex—-提取字符串的第0位置开始提取到index位置为止，并且不包括index位

NSString *string07;

string07 =［string01 substringToIndex:3］;

NSLog(@”string07 = %@”,string07);

9.提取字符串－－－－substringWithRang:NSMakeRange()—-从字符串的第6位置开始提取长度为5的字符串

NSString *string08;

string06 =［string01 substringWithRang:NSMakeRang(6,5)］;

NSLog(@”string08 = %@”,string08);

10.求文件的扩展名

NSString filename03 =@”~/hello.txt”;

NSLog(@”filename03 文件的扩展名是：%@”,[filename03 pathExtension]); //extension,延长；expand,扩展

11.文件的相对路径 转 为绝对路径－－－stringByExpandingTildeInPath——-

NSString *path01 = @”~/hello.txt”;

NSString *absPath = [path01 stringByExpandingTildeInPath];

NSLog(@”absPath = %@”,absPath);

12.文件的绝对路径 转 为相对路径－－－stringByAbbreviatingWithTildeInPath——-

NSString *path02 = @”/Users/qingyun/hello.txt”;

NSString *Path = [path02 stringByAnnreviatingWithTildeInPath];

NSLog(@”Path = %@”,Path02);

MAC上搭建hexo方法

一:

1. 安装nvm(node.js的版本管理工具）

2.

3.

4. $ wget -q0- https://raw.github.com/creationix/nvm/master/install.sh | sh

5.

6.

7. 添加如下内容到.zshrc配置文件

8.

9.

10. [ -s “/Users/users/.nvm/nvm.sh” ] && . “/Users/users/.nvm/nvm.sh” # This loads nvm

11.

12.

13. 安装node.js(通过nvm安装)

14.

15.

16. $ nvm install 0.10

17.

18.

19. 安装hexo（通过npm安装）

20. 1

21.

22. $ npm install -g hexo

23.

24.

25. 创建hexo文件夹，并安装hexo相关组件

26.

27.

28.

29.

30. $ hexo init hexo

31. $ cd hexo

32. $ npm install

33.

二.往自己blog 上传blog

1.在hexo目录下输入：hexo new 春暖花开，就创建了一个“ 春暖花开”的文件，然后用文本编辑器编辑。

如果创建成功则会提示存在hexo/source/_posts目录下

2.hexo目录下：hexo g //将文本转换为静态网页

3.hexo d //上传至github server

4. 访问自己博客即可看到效果
